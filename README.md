### Installation

1.  **Clone the repo**
    ```sh
    git clone [https://github.com/your-username/nutrisync-app.git](https://github.com/your-username/nutrisync-app.git)
    ```
2.  **Navigate to the project directory**
    ```sh
    cd nutrisync-app
    ```
3.  **Install dependencies**
    ```sh
    flutter pub get
    ```
4.  **Run the app**
    ```sh
    flutter run
    ```
